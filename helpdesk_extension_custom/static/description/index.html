Hold status for Helpdesk tickets with custom added reasons.

This module helps to mark the helpdesk tickets to hold state, with custom reasons.

1. Enterprise Version

2. Odoo.sh

Screenshots

1)Users can create custom reasons for putting tickets on hold state:

2) User can then select the helpdesk ticket which need to put on hold state and click on the 'Put on-Hold' button.

3) The button will open a popup, from there user can select the reasons from the dropdown list and click on 'Mark as on Hold' button to confirm.:

4) This will move the helpdesk into hold state and mark the hold wizard applied field:
